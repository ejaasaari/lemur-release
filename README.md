# lemur

Fast MaxSim-based retrieval with a small neural feature model.

## Requirements

- AVX-512 compatible CPU (required for the C++ extension)
- Python 3.11+ (recommended)

## Installation

From the repo root:

```bash
pip install .
```

## Example usage

Use the provided example script as-is:

```bash
python example.py
```

If you want to adapt it, start by copying `example.py` and editing it for your data.
